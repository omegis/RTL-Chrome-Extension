Rotem Daily RTL - Privacy Policy

Last Updated: September 30, 2025 | Version 2.4.4

Summary: Rotem Daily RTL does not collect, store, or transmit any user data. This extension operates entirely locally in your browser.

1. Information Collection

Rotem Daily RTL does NOT collect any personal information.

The extension does not:

  • Collect browsing history
  • Track user behavior
  • Access website content or user input
  • Store passwords, credentials, or personal data
  • Use cookies or tracking technologies
  • Send any data to external servers

2. Data Storage

The extension uses Chrome's local storage API to save one single preference:

  • Extension Enable/Disable State - A simple boolean (true/false) that remembers if you want the extension active or disabled.

This preference is stored locally on your device only and is never transmitted anywhere.

3. Permissions Explanation

The extension requires the following permissions for its core functionality:

Host Permissions (Specific Websites)

  • notion.so - To apply RTL styling to Hebrew text in Notion
  • claude.ai - To apply RTL styling to Hebrew chat messages
  • gemini.google.com - To apply RTL styling in Gemini Canvas
  • dash.bunny.net - To apply RTL styling in Bunny.net forms
  • app.manychat.com - To apply RTL styling in ManyChat builders

What the extension does: Only modifies CSS properties (direction, text-align, padding) to display Hebrew text correctly. Does not read, modify, or access any page content or user data.

Storage Permission

Used only to save your enable/disable preference locally.

Tabs Permission

Used only to detect which website you're currently on (to apply the correct styling). Does not access tab content, URLs, or browsing history.

4. Data Sharing

Rotem Daily RTL does NOT share any data with third parties because it doesn't collect any data in the first place.

  • No analytics services
  • No advertising networks
  • No external servers or APIs
  • No data transmission of any kind

5. How the Extension Works

The extension operates entirely locally in your browser:

  1. Detects Hebrew characters using Unicode pattern matching
  2. Applies CSS styling (direction: rtl, text-align: right) to elements containing Hebrew
  3. All processing happens in your browser's memory
  4. No data leaves your device

6. Security

Since the extension doesn't collect or transmit any data, there is no data to secure or breach. The extension:

  • Contains no external dependencies
  • Loads no remote code
  • Makes no network requests
  • Operates in a sandboxed browser environment

7. Children's Privacy

The extension does not collect data from anyone, including children under 13.

8. Changes to This Policy

If we ever change how the extension handles data (which we don't currently do), we will update this policy and notify users through the Chrome Web Store listing.

9. Open Source

The extension's source code is available on GitHub for full transparency:

https://github.com/omegis/RTL-Chrome-Extension

10. Contact

For questions about this privacy policy or the extension:

  • GitHub Issues: Report an issue
  • GitHub Repository: View source code
Bottom Line: Rotem Daily RTL is a simple, privacy-focused tool that only styles text. It doesn't know who you are, what you're doing, or where you're browsing. Your privacy is 100% protected.
